# 🚀 Portofolio Pribadi - Alfian Indra

Selamat datang di repositori portofolio saya! Projek ini dibangun menggunakan **Next.js** untuk menampilkan keahlian, pengalaman, dan projek-projek yang telah saya kerjakan.

## ✨ Fitur Utama

- **Responsive Design**: Tampilan yang optimal di berbagai perangkat (Mobile, Tablet, Desktop).
- **Modern UI/UX**: Menggunakan teknologi terbaru untuk pengalaman pengguna yang mulus.
- **Optimized Performance**: Kecepatan akses yang tinggi berkat optimasi Next.js.

## 🛠️ Teknologi yang Digunakan

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Font**: Geist (Optimized by `next/font`)
- **Deployment**: [Vercel](https://vercel.com/)
